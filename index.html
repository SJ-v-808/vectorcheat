vectors

What is vector ?

Vector is a quantity that has both magnitube and direction.

unit vector a^ = a/|a|

Magnitude |a| = sqrt(x2+y2+z2)

Dot Product a.b = |a||b|cosθ

Cross Product axb = |a||b|sinθ

What is linear combination of vectors?

A vector 'r' is said to be linear combination of vectors if there exist some scalar α,β and γ such that 'r' = αa+βb+γc

What is linearly dependent vector?

a,b and c vectors are said to be linear dependent vector if there exist scalar α,β and γ(not all zero) such that αa+βb+γc = 0

What is linearly independent vector?

a,b and c vectors are said to be linear independent vector if there exist scalar α,β and γ(all zero) such that αa+βb+γc = 0 , α=β=γ=0

Volume of parallelopiped or Scalar triple product = a.(bxc)

Vector triple product = ax(bxc)
